Introduction.

DRM is user-space wrapper for kernel drivers in Linux. In this repo, we write examples using drm API for amdgpu and radeon backends.

Installation

The repo is tested on Ubuntu 16.04 x86-64. Prereqs: `sudo apt-get install autoconf xutils-dev libtool libpthread-stubs`
